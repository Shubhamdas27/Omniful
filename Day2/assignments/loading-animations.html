1. Bouncing Dots

Classic three-dot loading animation with staggered bounce effect.

.dots-loader .dot { animation: bounce-dots 1.4s ease-in-out infinite both; } .dot:nth-child(1) { animation-delay: -0.32s; } .dot:nth-child(2) { animation-delay: -0.16s; } @keyframes bounce-dots { 0%, 80%, 100% { transform: scale(0); } 40% { transform: scale(1); } }

2. Rotating Ring

Simple rotating border spinner - most commonly used loading indicator.

.ring-loader { border: 5px solid #e2e8f0; border-top: 5px solid #667eea; border-radius: 50%; animation: spin 1s linear infinite; } @keyframes spin { 100% { transform: rotate(360deg); } }

3. Pulsing Circle

Expanding circle with fade-out effect for subtle loading indication.

.pulse-loader { animation: pulse 1.5s ease-in-out infinite; } @keyframes pulse { 0% { transform: scale(0); opacity: 1; } 100% { transform: scale(1); opacity: 0; } }

4. Wave Bars

Audio wave-style loader with animated bars of different heights.

.wave-loader .bar { animation: wave 1.2s ease-in-out infinite; } .bar:nth-child(1) { animation-delay: 0s; } .bar:nth-child(2) { animation-delay: 0.1s; } @keyframes wave { 0%, 40%, 100% { transform: scaleY(0.4); } 20% { transform: scaleY(1); } }

5. Square Flip

3D flip animation using CSS transforms for modern loading effect.

.flip-loader { animation: flip 1.2s ease-in-out infinite; } @keyframes flip { 0%, 50% { transform: rotateY(0deg); } 50.01%, 100% { transform: rotateY(180deg); } }

6. Typing Indicator

Chat-style typing indicator with bouncing dots animation.

.typing-dot { animation: typing 1.4s infinite ease-in-out; } .typing-dot:nth-child(1) { animation-delay: 0s; } .typing-dot:nth-child(2) { animation-delay: 0.2s; } @keyframes typing { 0%, 60%, 100% { transform: translateY(0); } 30% { transform: translateY(-15px); } }

7. Growing Circle

Expanding ring animation with rotation and scale effects.

.grow-loader::before { border-top: 3px solid #667eea; animation: grow 2s linear infinite; } @keyframes grow { 0% { transform: rotate(0deg) scale(0); } 100% { transform: rotate(360deg) scale(1); opacity: 0; } }

8. DNA Helix

Complex organic movement pattern simulating DNA strand rotation.

.strand { animation: dna-rotate 2s linear infinite; } .strand:nth-child(1) { animation-delay: 0s; } .strand:nth-child(2) { animation-delay: 0.25s; } @keyframes dna-rotate { 0% { top: 26px; left: 26px; transform: scale(0); } 25% { top: 10px; left: 45px; transform: scale(1); } }

9. Orbit System

Multiple rotating rings creating a planetary orbit effect.

.orbit-ring { border-top: 2px solid #667eea; animation: orbit 1.5s linear infinite; } .orbit-ring:nth-child(2) { animation-delay: 0.5s; } @keyframes orbit { 100% { transform: rotate(360deg); } }

âš¡ Performance Optimization Tips

  • Use transform and opacity: These properties are GPU-accelerated and provide smooth animations
  • Add will-change: Use will-change: transform to hint the browser about upcoming animations
  • Avoid animating layout properties: Don't animate width, height, or position properties
  • Use transform3d(): Enable hardware acceleration with transform: translateZ(0)
  • Optimize timing functions: Use ease-out or custom cubic-bezier for natural motion
  • Consider reduced motion: Respect user preferences with @media (prefers-reduced-motion)
